# TOU Connect - 東京通信大学 学友マッチングプラットフォーム

## 概要
東京通信大学の学生同士のつながりを促進するAI活用マッチングサービス

## 機能
- X(Twitter)投稿分析による自動プロフィール生成
- 学習分野・興味関心の可視化
- 学友マッチング・推薦システム
- コミュニティ形成支援

## 技術スタック
- Python + Flask
- Gemini API (AI分析)
- Twitter API (データ取得)
- SQLite (データベース)
- D3.js (可視化)

## 開発フェーズ
1. Phase 1: X投稿分析システム
2. Phase 2: マッチング機能
3. Phase 3: 可視化・UI
4. Phase 4: コミュニティ機能
