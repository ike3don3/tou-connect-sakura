## 🚀 ローンチ手順

### Step 1: 本番環境準備（30分）
```bash
# サーバー準備
./deploy.sh production

# Docker 起動
docker-compose up -d

# SSL証明書取得
certbot --nginx -d touconnect.com
```

### Step 2: 動作確認（15分）
```bash
# ヘルスチェック
curl https://touconnect.com/health

# 機能テスト
curl -X POST https://touconnect.com/api/analyze \
  -H "Content-Type: application/json" \
  -d '{"username": "testuser"}'
```

### Step 3: 監視設定（10分）
